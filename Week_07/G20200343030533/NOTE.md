```c
x ^ 0 = x
x ^ 1s = ~x
x ^ ~x = 1s //得到一个全1二进制
x ^ x = 0
c = a ^ b; b = a ^ c , a = b ^ c 
a^b^c = (a^b)^c = a^(b^c)
```

操作指定位 

```c
x & (~0<<n) ; // 右边n位清零
(x>>n)&1 ;//获取第n位
x & ( 1 <<(n-1)) 
x | ( 1 << n); // 将第n位变1
x & (~(1<<n)); //将第n位变0
x & ( (1<<n) -1)
x & ( ~( ( 1 << ( n + 1)) -1 ) )
```
