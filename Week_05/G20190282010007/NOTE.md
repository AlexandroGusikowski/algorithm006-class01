动态规划篇
    动态规划和递归、分治并没有根本上的区别（关键在于看是否有最优子结构）
    共性：找到重复子问题
    差异性： 最优子结构、中途可以淘汰次优解

斐波拉契数列的第N项
    公式： f(n) = f(n-1) + f(n-2)

    暴力递归
    f（n） {
        return n <= 1 ？ n ：f(n-1) + f(n-2)
    }

    优化解
    f (n, array:[]) {
        if (n <= 1) {
            return n
        }
        if (array[n] == 0) {
            array[n] = array[n-1] + array[n-2]
        }
        return array[n]
    }

    优化解2
    a[0] = 0;
    a[1] = 1;
    for (i=2; i<=n; i++) {
        a[i] = a[i-1] + a[i-2]
    }
    a[n]
